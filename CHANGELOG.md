# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-26

### Added
- Created portfolio layout structure featuring Hero, About, Skills, Projects, Certifications, and Contact sections.
- Configured a dynamic mouse cursor layer and magnetic button elements.
- Embedded featured project case study slider overlay drawer detailing SecureU and AuraBeauty AI.
- Added simulated GitHub contributions calendar.
- Integrated credentials validation trigger mechanisms.
- Established a complete, responsive visual look tracking a warm beige, charcoal, and gold editorial styling palette.
- Created `LICENSE`, `CONTRIBUTING.md`, and default git configurations.

### Fixed
- Cleaned unused imports and unescaped HTML characters inside React JSX tags.
- Fixed TypeScript type constraints across custom hooks and projects.
- Resolved synchronous render warnings inside the mouse tracking loops to support React 19 compatibility.
